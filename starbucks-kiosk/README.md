# 프로젝트 설명

이 프로젝트는 2021년 학원 시절 포트폴리오를 `Next.js`와 `TypeScript`를 사용하여
리팩토링 한 프로젝트입니다.

## Getting Started

```bash
npm run dev
# or
yarn dev
```

## 폴더 구조

```bash
about-me
│
├── 📂app/
│   ├─── 📂_api/                            # 공통 API
│   │   ├─── 🗒️test.ts                      # ex: 사용자 데이터 호출 등
--------------------------------------------------------------------------
│   ├─── 📂_constants/                      # 전역에서 사용되는 상수 값
--------------------------------------------------------------------------
│   ├─── 📂_hooks/                          # 공통 커스텀 훅
│   │   ├─── 🗒️useTest.ts
│   │   └─── 🗒️useTesT.ts
--------------------------------------------------------------------------
│   ├─── 📂_types/                          # 타입 정의
│   │    ├─── 🗒️test.ts
│   │    └─── 🗒️test.ts
--------------------------------------------------------------------------
│   ├─── 📂_utils/                          # 유틸리티 함수 모음
│   │    └─── 🗒️helpers.ts                  # 다양한 유틸 함수
--------------------------------------------------------------------------
│   ├─── 📂_ui/                             # UI 관련 요소 모음
│   │   ├── 📂components/                   # 공통 컴포넌트 (Button, Modal 등)
│   │   │   ├─── 📂button/
│   │   │   │   ├── 🗒️Button.tsx
│   │   │   │   └── 🗒️LoginButton.tsx
│   │   │   └─── 📂modal/
│   │   │   │   ├── 🗒️Alert.tsx
│   │   │   │   └── 🗒️Confirm.tsx
│   │   ├─── 📂context/                     # Context 관리
│   │   │   ├── 🗒️TestContext.tsx
│   │   │   └── 🗒️TestContext.tsx
│   │   ├─── 📂styles/                      # 스타일 관리
│   │   │   └── 🗒️global.css                # 전역 스타일
--------------------------------------------------------------------------
│   ├─── 📂(home)/                          # 메인 페이지
│   │   └─── 🗒️page.tsx/
│   ├─── 📂(pages)/                         # 페이지들
│   └── 🗒️layout.tsx                        # 공통 레이아웃 설정
--------------------------------------------------------------------------
└─── 📂public/                              # 정적 파일 (이미지, 아이콘 등)
     ├─── 📂images/
     └─── 📂fonts/
```
